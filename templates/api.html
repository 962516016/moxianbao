使用方法简介

longyuanpower是风电功率预测与分析的Python SDK客户端，为使用风电功率预测与分析的开发人员提供了一系列的交互方法。 参考如下代码
新建一个longyuanpower.py： 

from longyuanpower import longyuanapi
API_URL = 'http://172.24.187.133:5446/longyuanapi'
API_KEY = '你的 Api Key'
SHOW = True
OUTPUT = True
TRAIN= True
filepath = '你的 文档路径'
longyuanapi(api_url, api_key, show, output, train).savepredictfile(filepath)

在上面代码中，API_KEY是系统分配给用户的，为字符串，用于标识用户，为访问做签名验证，可在个人中心查看。
